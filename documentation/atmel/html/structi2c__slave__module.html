Detailed Description

SERCOM I2C Slave driver software device instance structure.

SERCOM I2C Slave driver software instance structure, used to retain software state information of an associated hardware module instance.

Note:
The fields of this structure should not be altered by the user application; they are reserved for module-internal use only.
