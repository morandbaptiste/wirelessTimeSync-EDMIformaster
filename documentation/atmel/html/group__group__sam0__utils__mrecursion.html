#define MRECURSION (   count,

)    TPASTE2(MRECURSION, count) (macro, data)

Macro recursion.

This macro represents a horizontal repetition construct.

Parameters:
[in]countThe number of repetitious calls to macro. Valid values range from 0 to MRECURSION_LIMIT.
[in]macroA binary operation of the form macro(data, n). This macro is expanded by MRECURSION with the current repetition number and the auxiliary data argument.
[in]dataA recursive threshold, building on this to decline by times defined with param count.
Returns:
macro(data-count+1,0) macro(data-count+2,1)...macro(data,count-1)
#define MRECURSION_LIMIT   256

Maximal number of repetitions supported by MRECURSION.
